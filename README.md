# SAWM — Benchmark

Carte de notation de l'agent **SAWM** : mesures réelles exécutées sur l'agent en production.

- **Volet système** — adaptation de langue (12 langues), routage d'outils, verrous de sécurité, latence, fiabilité.
- **Volet raisonnement** — 14 domaines (mathématiques, finance, sentiment, enquête, logique, mémoire, créativité, histoire, biochimie, proposition, mission, réflexion, défense analytique). Vérification objective quand la réponse est connue, sinon jury neutre.

Chaque score est reproductible. Voir [`index.html`](./index.html) (page en ligne via GitHub Pages).

> Méthode & limites détaillées en bas de la carte. Les problèmes mathématiques *ouverts* ne sont jamais présentés comme « résolus » ; la « défense » est strictement analytique/défensive.
